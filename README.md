# Activity - Deploy your Flask app to a cloud host
1. Initialise a git repo in your Flask app (app3.py)
2. Create an empty repo in GitHub
3. Push your Flask app to the GitHub repo
4. Deploy your Flask app to a cloud host via the GitHub repo - see https://render.com/docs/deploy-flask
   Render will: Read the GitHub repo, install dependencies, start the Flask server, Give a public URL
